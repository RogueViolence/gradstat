The Independent t-test

Comparing two groups? This is your test. Let's start with a real scenario you'll actually recognize.

1
Real example first
Picture this
You're studying whether a new CBT intervention reduces anxiety scores. You have two groups — one that received CBT and one that didn't. Both groups completed the GAD-7 anxiety scale at the end of the study. You want to know: did the CBT group score meaningfully lower than the control group?

So what are we actually asking?

You have two separate groups of people, and you measured one thing (anxiety scores). The question is simple: are the average scores different enough between the two groups that it's probably not just chance?

That's it. That's all a t-test is asking. "Are these two group averages far enough apart that something real is probably going on?"

What does the t-test actually do?

It looks at the difference between your two group means and compares that difference to how much natural variability exists in your data. If the difference is big relative to the variability — boom, statistically significant.

The output gives you a t-value and a p-value. The p-value is what you care about most — if it's below .05, you have a statistically significant difference.

2
When to use it
Groups
Exactly 2 independent groups
Variable type
Continuous outcome (scores, measures)
Relationship
Groups are unrelated (different people)
Question
"Is there a difference between these two groups?"
✓ Use it when
  • Comparing treatment vs control
  • Male vs female on a scale
  • Two different schools/sites
  • Intervention vs no intervention
✗ Don't use it when
  • You have 3+ groups (use ANOVA)
  • Same people measured twice (use paired t-test)
  • Your outcome is categorical (use chi-square)
  • You want to predict, not compare

3
Running it in SPSS & R

Your data needs two columns — one for the group (coded as numbers, like 0 and 1) and one for the score. Like this:

/* In SPSS, use the menu: */ Analyze → Compare Means → Independent-Samples T Test /* Set up: */ Test Variable(s): anxiety_score Grouping Variable: group Define Groups: Group 1 = 0, Group 2 = 1 /* Click OK and look for: */ - Levene's Test (check equal variances) - t-value, df, Sig. (2-tailed) ← your p-value
# Load your data first data <- read.csv("mydata.csv") # Run the t-test t.test(anxiety_score ~ group, data = data, var.equal = TRUE) # Output shows: # t = your t-value # df = degrees of freedom # p-value = your significance # 95% CI = confidence interval

Reading the output

The most important numbers: your t-value, degrees of freedom (df), and p-value. If p < .05, the difference is statistically significant. Also check your group means — a significant result with a tiny mean difference might not be practically meaningful.

Reporting in APA format

APA format for a t-test result follows a specific pattern. Here's the template and a filled-in example:

APA Template
A(n) [type] t-test was conducted to compare [outcome] in [group 1] and [group 2] conditions. There was a [significant/non-significant] difference in scores for [group 1] (M = ___, SD = ___) and [group 2] (M = ___, SD = ___) conditions; t(df) = ___, p = ___.
Filled-in Example
An independent samples t-test was conducted to compare anxiety scores in CBT and control conditions. There was a significant difference in scores for the CBT group (M = 8.2, SD = 2.1) and the control group (M = 12.4, SD = 2.8); t(48) = 6.34, p < .001.
Note: Report exact p-values when possible (e.g., p = .032). Only use "p < .001" when the value is smaller than .001. Always report means and standard deviations.

Lesson 2 of 3

One-Way ANOVA

Like a t-test, but for when you have three or more groups. More groups, more power — but also more to interpret.

1
Real example first
Picture this
You're comparing depression scores across three treatment conditions — CBT, medication, and a waitlist control. You measured depression with the PHQ-9 after 8 weeks. You want to know: do any of the three groups differ significantly from each other?

Why not just run three t-tests?

Great question — and this is exactly what trips people up. If you run three separate t-tests (CBT vs. med, CBT vs. control, med vs. control), you inflate your chance of a false positive. Every test has a 5% chance of a false positive, and those add up fast.

ANOVA tests all three groups simultaneously in one test, keeping your error rate honest at 5%.

What does ANOVA actually do?

It compares the variance between your groups to the variance within your groups. If the groups are more different from each other than people are different within the same group — that's a signal something real is going on.

The output gives you an F-value and a p-value. But here's the catch — a significant ANOVA just tells you that a difference exists somewhere. It doesn't tell you which groups differ. That's what post-hoc tests are for.

1 of 4
2
When to use it
Groups
3 or more independent groups
Variable type
Continuous outcome (scores, measures)
Relationship
Groups are independent (different people)
Follow-up
Post-hoc tests (Tukey, Bonferroni) to find which groups differ
✓ Use it when
  • 3+ treatment conditions
  • Multiple demographic groups
  • Comparing multiple schools/sites
  • 3+ dose levels of an intervention
✗ Don't use it when
  • You only have 2 groups (use t-test)
  • Same people across conditions (use repeated measures ANOVA)
  • Multiple outcome variables (use MANOVA)
  • You want to control for a covariate (use ANCOVA)

3
Running it in SPSS & R
/* In SPSS, use the menu: */ Analyze → Compare Means → One-Way ANOVA /* Set up: */ Dependent List: depression_score Factor: treatment_group /* Click Post Hoc → check Tukey */ /* Click Options → check Descriptive */ /* Click OK */ /* Look for: */ - F-value and Sig. in ANOVA table - Multiple Comparisons table (post-hoc)
# Run one-way ANOVA model <- aov(depression_score ~ treatment_group, data = data) summary(model) # Post-hoc Tukey test TukeyHSD(model) # Output shows: # F value = your F-statistic # Pr(>F) = your p-value # Tukey: which pairs differ

Don't skip the post-hoc tests

A significant F just means something is different somewhere. Run Tukey's HSD (honestly significant difference) to find out which specific groups differ from each other. That's usually what your research question actually needs.

4
Reporting in APA format
APA Template
A one-way ANOVA was conducted to compare [outcome] across [groups]. There was a [significant/non-significant] effect of [group variable] on [outcome], F(df_between, df_within) = ___, p = ___. Post hoc comparisons using Tukey's HSD indicated that [group A] (M = ___, SD = ___) differed significantly from [group B] (M = ___, SD = ___).
Filled-in Example
A one-way ANOVA was conducted to compare depression scores across CBT, medication, and waitlist control conditions. There was a significant effect of treatment condition on depression scores, F(2, 87) = 14.32, p < .001. Post hoc comparisons using Tukey's HSD indicated that the CBT group (M = 7.4, SD = 2.3) and the medication group (M = 8.1, SD = 2.7) both scored significantly lower than the waitlist control group (M = 13.6, SD = 3.1), p < .001.
Note: The two numbers in F(2, 87) are degrees of freedom between groups and within groups. Both come straight from your ANOVA table output.

Lesson 3 of 3

Linear Regression

When you want to predict one thing from another — or understand what's driving an outcome. This is where stats gets really useful.

1
Real example first
Picture this
You want to know if the number of therapy sessions someone attends predicts their depression score at the end of treatment. More sessions → lower depression? You have both variables measured for 60 participants and you want to quantify that relationship.

What does regression actually do?

It draws the best possible straight line through your data that predicts your outcome variable (depression score) from your predictor variable (number of sessions). That line has a slope — and the slope tells you how much the outcome changes for every one-unit increase in the predictor.

It also tells you R² — how much of the variation in your outcome is explained by your predictor. An R² of .40 means your predictor explains 40% of the variance in the outcome. That's actually pretty good in psychology research.

Simple vs. multiple regression

Simple regression = one predictor. Multiple regression = two or more predictors (e.g., sessions + baseline severity → depression score). Multiple regression is what you'll use most in research — it lets you control for confounds.

2
When to use it
Goal
Predict or explain an outcome
Outcome variable
Continuous (scores, measures)
Predictors
Continuous or categorical (coded)
Key output
β coefficients, R², p-values
✓ Use it when
  • Predicting an outcome from variables
  • Controlling for confounds
  • Testing if X predicts Y
  • Exploring relationships between variables
✗ Don't use it when
  • Your outcome is categorical (use logistic regression)
  • You just want to compare group means (use t-test/ANOVA)
  • You have non-linear relationships
  • Data violates normality badly
2 of 4
3
Running it in SPSS & R
/* In SPSS, use the menu: */ Analyze → Regression → Linear /* Set up: */ Dependent: depression_score Independent(s): num_sessions /* Click Statistics → check: */ /* Estimates, Confidence intervals, R squared */ /* Click OK */ /* Look for: */ - Model Summary: R, R², Adjusted R² - ANOVA table: F and Sig. - Coefficients: B, Beta, t, Sig.
# Run simple linear regression model <- lm(depression_score ~ num_sessions, data = data) summary(model) # Multiple regression (add predictors) model2 <- lm(depression_score ~ num_sessions + baseline_severity, data = data) summary(model2) # Output shows: # Coefficients: Estimate (B), p-value # Multiple R-squared: variance explained # F-statistic: overall model significance

Reading the output

Focus on three things: R² (how much variance is explained), the F-test p-value (is the overall model significant?), and the β coefficients with their p-values (which predictors are significant, and in which direction?).

A negative β means more of that predictor = lower outcome. A positive β means more = higher. The size of β tells you the magnitude of the effect.

3 of 4
4
Reporting in APA format
APA Template
A simple linear regression was performed to predict [outcome] from [predictor]. The results of the regression indicated that [predictor] significantly predicted [outcome], F(df1, df2) = ___, p = ___, R² = ___. [Predictor] was a [positive/negative] predictor of [outcome] (β = ___, p = ___).
Filled-in Example
A simple linear regression was performed to predict depression scores from number of therapy sessions attended. The results indicated that session attendance significantly predicted depression scores, F(1, 58) = 18.74, p < .001, R² = .24. Number of sessions was a significant negative predictor of depression scores (β = −.49, p < .001), indicating that greater session attendance was associated with lower depression scores.
Note: Report standardized β (Beta) for comparing predictors, or unstandardized B with SE when the raw scale matters. R² tells your reader how much variance your model explains — always include it.
